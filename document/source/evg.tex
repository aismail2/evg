\section{Requirements}
The driver should support the following features:
\begin{itemize}
	\item Event sequencer
	\item Software events.
	\item Multiplexed counters
	\item AC line synchronization
	\item Event clock
\end{itemize}

	\subsection{Event Sequencer}
	\begin{itemize}
		\item Control each RAM separately.
		\item Enable/disable RAM. This sets the RAM in automatic recycle on trigger.
		\item Program event code with timestamp at RAM address.
		\item Run from the event clock and program the sequencer clock prescaler.
		\item Trigger sequencer from AC mains logic or by software.
	\end{itemize}

	\subsection{Software Event}
	\begin{itemize}
		\item Enable/disable software event.
		\item Set software event.
		\item Trigger software event.
	\end{itemize}

	\subsection{Multiplexed Counters}
	\begin{itemize}
		\item Set prescaler to each multiplexed counter.
	\end{itemize}

	\subsection{AC Line Synchronization}
	\begin{itemize}
		\item Enable triggering of the sequencer.
		\item Set prescaler.
		\item Select synchronization signal.
	\end{itemize}

	\subsection{Event Clock}
	\begin{itemize}
		\item Set RF prescaler.
		\item Set clock divider.
	\end{itemize}

	\subsection{Misc.}
	\begin{itemize}
		\item Read firmware version.
		\item Enable/disable the card.
		\item 
	\end{itemize}
